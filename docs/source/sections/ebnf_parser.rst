EBNF / Parser
=============

The following tools are available for working with ASSERTL:

- :download:`ASSERTL grammar <assertl.lark>` in EBNF for `Lark <https://lark-parser.readthedocs.io/en/latest/grammar.html>`__ .

- :download:`ASSERTL parser <parser.py>` using Lark.
  To run the tests:

    .. code-block:: bash
    
        pip install lark
        python parser.py
